## 🚀 Featured Projects

### 🛠️ [Resume Builder SaaS (AI Resume Builder)](https://github.com/deepakkandpal004/Resume-Builder-SaaS)
*An AI-powered resume creation and optimization platform.*
* **AI-Assisted Writing**: Integrated Google Gemini API for context-based resume generation and prompt-driven optimizations.
* **Dynamic Image Toolkit**: Configured ImageKit API for profile image uploads and automated background removals.
* **Authentication**: Implemented secure JWT user sessions with custom authorization middlewares.
* **Tech Stack**: React.js, Node.js, Express.js, MongoDB, JWT, Google Gemini API, ImageKit, Tailwind CSS.

### 💰 [AI-Powered Expense Tracker](https://github.com/deepakkandpal004/next-expense-tracker)
*A smart personal finance tracker that categorizes expenses and provides budget insights using LLMs.*
* **Conversational AI Analysis**: Integrated OpenRouter API to deliver dynamic spending analysis and personalized budget advice.
* **Fast Database Operations**: Neon PostgreSQL backend setup for fast, serverless transactional queries.
* **User Authentication**: Integrated Clerk authentication for secure signups, log-ins, and page routing.
* **Tech Stack**: Next.js (React), Neon PostgreSQL, Clerk Authentication, OpenRouter API, Tailwind CSS, Vercel.

### 🌐 [Interactive 3D Developer Portfolio](https://github.com/deepakkandpal004/Portfolio-website)
*A highly interactive portfolio showcasing a modern, polished personal brand.*
* **Immersive Experience**: Built with Three.js/React Three Fiber for interactive 3D web assets.
* **High-fidelity Animations**: Smooth page transitions, slider menus, and viewport-scroll timelines powered by GSAP.
* **Tech Stack**: React, TypeScript, Three.js, GSAP, Tailwind CSS.

### 🔄 [P2P File Sharing Platform](https://github.com/deepakkandpal004/P2P-File-Sharing)
*A decentralized web application for direct peer-to-peer file transfers.*
* **Brokerless Direct Transfer**: Built a chunk-based direct file transfer mechanism supporting massive files without server storage overhead using PeerJS (WebRTC).
* **Signaling Server**: Implemented a Socket.IO connection manager for peer-to-peer matchmaking and network fallback.
* **Tech Stack**: React.js, Node.js, Express.js, MongoDB, PeerJS, Socket.IO.

### 🧠 [Natural Language to Python Compiler](https://github.com/deepakkandpal004/NLP-Compiler)
*Rule-based natural language to Python compiler translating instructions.*
* **NLP Pipeline**: Used spaCy for text tokenization and syntax parsing, matching grammatical rules to generate clean, executable Python scripts.
* **Tech Stack**: Python, spaCy NLP.

### 🔗 [Secure URL Shortener](https://github.com/deepakkandpal004/URL-Shortener)
*A secure link shortening app with comprehensive tracking.*
* **Click Analytics**: Captures visitor user-agents, click counts, timestamps, and locations in PostgreSQL databases.
